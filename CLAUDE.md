# CLAUDE.md - Project Context for Claude Code

This file provides context for Claude Code when working on this project.

## Project Type

Static documentation website - GitHub Pages.

## Key Files

- `.claude/AGENTS.md` - Detailed agent instructions
- `.claude/rules/` - Project-specific rules
  - `theme-icon.md` - Theme and icon guidelines
  - `language.md` - Language guidelines
- `.claude/plan/` - Project plans (not tracked in git)

## Quick Start

1. Theme system: 26 themes controlled via `data-theme` attribute
2. Icons: Use simple-icons v3 via jsDelivr
3. Plans: Create in `.claude/plan/` directory

## Important Notes

- This is a static site - no build process needed
- All user replies should be in Chinese (unless requested otherwise)
- Plans go in `.claude/plan/` - these are not tracked in git
