Expression & Dialect

表达式收集语义，方言生成 SQL。

Expression Basics

表达式基础。

BaseExpression.to_sql(dialect) — 表达式委托方言生成 SQL。

expression_basics.py
from rhosocial.activerecord.backend.expression import Column, Literal

# 表达式收集语义参数，不含 SQL 细节
name = Column('username')
age = Column('age')

# 运算符重载构建条件
condition = (name == 'Alice') & (age > 18)
# → 委托 dialect.format_*() 生成目标 SQL
sql, params = condition.to_sql(dialect)

Predicates

谓词体系。

predicates.py
from rhosocial.activerecord.backend.expression import (
    InPredicate, BetweenPredicate,
    LikePredicate, IsNullPredicate,
    ComparisonPredicate
)

# ComparisonPredicate
cond = ComparisonPredicate(Column('age'), '>', Literal(18))

# InPredicate
cond = InPredicate(Column('category'), ['A', 'B'])

# BetweenPredicate
cond = BetweenPredicate(Column('stock'), 10, 100)

# IsNullPredicate
cond = IsNullPredicate(Column('deleted_at'))

Advanced

CASE WHEN 示例。

case_expression.py
from rhosocial.activerecord.backend.expression import CaseExpression, Literal

# CASE WHEN active = True THEN 'Active' ELSE 'Inactive' END
status_label = CaseExpression() \
    .when(User.c.active == True, Literal('Active')) \
    .else_(Literal('Inactive'))
DQL

SELECT 语句。

query_expression.py
from rhosocial.activerecord.backend.expression import QueryExpression

# QueryExpression — 完整 SELECT 构建
query = QueryExpression() \
    .select_from('users') \
    .where(User.c.age > 18) \
    .order_by(User.c.created_at.desc()) \
    .limit(10) \
    .offset(20)

sql, params = query.to_sql(backend.dialect)
